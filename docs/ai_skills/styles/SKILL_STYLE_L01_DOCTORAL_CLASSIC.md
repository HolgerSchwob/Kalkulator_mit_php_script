# Style: L-01 — Doctoral Classic

**Technik:** `SKILL_01_TECHNICAL.md` + `SSOT_SVG_COVER_TEMPLATES.md`.

## Visuelle DNA

- **Symmetrisch**, oft **vertikal zentriert** auf U1.
- **Serif** für den Haupttitel (Eleganz, Tradition); Sans für Nebentext möglich.
- **Zarte Divider** oder feine Linien; viel **Weißraum**.
- Würdevoll, akademisch, zeitlos.

## Typografie

- Titel: Serif mit starkem Kontrast zu Admin-Zeilen (Playfair / ähnlich + Lato — siehe SKILL_02 global).

## Layout

- U1: zentrierte oder symmetrische Achse; ausreichend Abstand zum Falz (x ≥ 290 für linksbündige Varianten beachten).

## Farbe

- Oft zurückhaltende Paletten (C-01, C-02); color1 für Flächen, color2 für Akzente und feine Linien.
