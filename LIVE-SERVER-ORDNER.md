# Live Server: richtiger Ordner

**Wichtig:** Live Server muss aus **diesem** Ordner gestartet werden: `Kalkulator 130226`.

- Im übergeordneten Ordner „Kalkulator in Bearbeitung“ liegt eine **andere** `index.html` (ca. 24 KB, ohne Kalkulator-/Modal-Script).
- Nur die `index.html` **hier** im Ordner `Kalkulator 130226` ist vollständig (ca. 116 KB, mit allem Script).

**So geht’s:** In Cursor/VS Code diesen Ordner `Kalkulator 130226` als Workspace öffnen, dann z. B. Rechtsklick auf `index.html` → „Open with Live Server“. Dann wird die richtige Seite ausgeliefert.
